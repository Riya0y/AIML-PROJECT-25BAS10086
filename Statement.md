**📌 Problem Statement:-**

In modern delivery systems, agents often struggle to find the most cost-efficient and time-saving route to a destination.
This challenge becomes even more complex in dynamic environments, where obstacles may appear or move unexpectedly. As a result, traditional static path planning methods fail to provide optimal solutions in real-time.


**🎯 Scope of the Project:-**

This project aims to:

1.Demonstrate and compare pathfinding algorithms such as BFS, UCS, and A*

2.Simulate environments with both static and dynamic obstacles

3.Showcase how an agent can adapt and replan its path in real-time

4.Analyze the efficiency, cost, and performance of each algorithm


**👥 Target Users:-**

This system can benefit:

1.🎓 Students – Learning AI and pathfinding concepts

2.🏢 Office Workers – Optimizing commute/navigation systems

3.🏠 Residents – Improving delivery efficiency for:
                     
                       .Food 🍔
                      
                       .Groceries 🛒
                     
                       .Documents 📦


**🚀 High-Level Features:-**

1️⃣.Grid World Simulation

1.1:Creates a virtual 2D environment

1.2:Includes different terrains and obstacles

2️⃣ Dynamic Obstacles

2.1:Adds moving blocks/enemies

2.2:Simulates real-world unpredictability

3️⃣Intelligent Pathfinding

                    Implements:
                         °BFS (Breadth-First Search)
                         °UCS (Uniform Cost Search)
                         °A* (A-Star Algorithm)

4️⃣ Real-Time Adaptation

4.1:Agent detects blocked paths

4.2:Instantly recalculates a new optimal route

5️⃣ Algorithm Comparison

          Evaluates performance based on:
            ⏱️ Time taken
            💰 Path cost
            📊 Efficiency
**💡 Summary**

This project provides a practical implementation of AI-based navigation, helping understand how intelligent systems can make decisions in changing environments, just like real-world delivery and navigation systems.
